# YourUNet-YourSegNet-ACDC

## Installation
Installer les dépendances dans un environnement python
```
pip install -r requirements.txt
```
si vous travaillez sur votre ordinateur personnel équipé d’un bon GPU ou dans
l’environnement Colab comme vous l’avez fait pour les travaux pratiques antérieurs.  Une fois
votre environnement configuré, vous pouvez exécuter la commande
```
python train.py --help
```

pour voir les options s’offrant à vous.  Si vous exécutez la commande

```
python train.py --model UNet
```
