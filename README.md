# YourUNet-YourSegNet-ACDC
Segmentation 3D ciné-IRM cardiaque en 4 classes (Fond, VD, Myocarde, VG) sur la base de données ACDC via des architectures personnalisées (YourUNet &amp; YourSegNet) développées en PyTorch. Contient l'implémentation de fonctions de perte sur mesure, du checkpointing en temps réel et de l'augmentation de données en ligne de commande.
